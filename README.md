# Covenant Detail Co.

Premium mobile detailing website for Carmel, Indiana and Miami.

Live site: [covenantcardetailing.com](https://covenantcardetailing.com)

## Tech

- React 18
- Vite
- No external CSS — all inline (luxury serif typography via Google Fonts)

## Develop locally

```bash
npm install
npm run dev
```

## Deploy

Auto-deploys to Vercel on every push to `main`.
